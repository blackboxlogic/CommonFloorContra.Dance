Volunteering

This event can't run without volunteers. Please email CommonFloorContraDance@gmail.com to volunteer for the dance and indicate which role(s) you're interested in. Volunteering gets you free entry and a warm fuzzy feeling.

Positions:

  • Setup 6:30-7:00 (2 helpers: furniture, fans, signs)
  • Door shift A 6:45-7:45 (4 helpers: check vax/masks/CoC, shoes, money, stamp)
  • Door shift B 7:45-8:30 (2 helpers: same as above)
  • Door shift C 8:30-9:15 (1 helper: same as above)
  • Takedown 10-10:15 (2+ helpers: furniture, fans, signs)
